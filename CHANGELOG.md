# Changelog

## v1.0 – DiscoveryRank
Initial stable release of the DiscoveryRank recommendation-quality evaluation framework.

Key features:
- Modular pipeline for recommendation evaluation
- Time-safe candidate generation
- Multiple ranking strategies (popularity, freshness, diversity reranking)
- Advanced evaluation metrics (relevance, freshness, diversity, repetition, novelty, serendipity, coverage)
- Learned baseline using truncated SVD
- Reproducible notebook pipeline
