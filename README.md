# DiscoveryRank

A Python-first recommendation evaluation project.

## Directory Structure
- `data/`: Raw and processed data
- `notebooks/`: Jupyter notebooks for exploratory analysis
- `src/`: Source code modules
- `eval/`: Evaluation metrics and scripts
- `outputs/`: Output artifacts, reports, and models
- `docs/`: Documentation

## Setup
Create a virtual environment and `pip install -r requirements.txt`.
